Web Technologies

HTML

HTML (short for HyperText markup language) is a markup language used for writing out text and the how that text would be displayed on a website (the structure of the text)

CSS

CSS is a styling language that adds color to your code. It is commonly used on languages such as html.

JavaScript

JavaScript is a programming language used to give websites more interactive features (like buttons). Unlike Java, Javascript is more easier to understand and simplified.

Bootstrap

Bootstrap is a free workspace used for developing websites. It provides a framework for code and containds HTML, CSS and Javascript.
